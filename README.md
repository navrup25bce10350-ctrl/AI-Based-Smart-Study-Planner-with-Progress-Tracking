# AI-Based-Smart-Study-Planner-with-Progress-Tracking
##📌Overview

The Smart Study Planner is a simple Python-based application that helps students manage their study tasks efficiently. It generates a prioritized study schedule based on task deadlines and difficulty levels, and allows users to track their progress.

This project is developed as part of the Fundamentals of AI and ML (BYOP) course.

##🎯 Problem Statement

Students often struggle with:

Managing multiple subjects and deadlines
Prioritizing tasks effectively
Tracking study progress

This project provides a solution by automating study planning using basic AI concepts.

##🚀 Features
Add study tasks with deadline and difficulty
Automatically generate a prioritized study plan
View tasks sorted by urgency
Update task status (Pending → Completed)
Menu-driven interactive system
🧠 AI Concepts Used
Heuristic-based decision making
Priority-based scheduling
Basic optimization strategy

Priority Formula:

Priority = Difficulty / Days Remaining
